# 📟 Exemples Morse - Guide Pratique

## 🎯 Messages d'Urgence

### SOS (Détresse internationale)
```
... --- ...
S   O   S
```
**Comment le taper** :
1. Tap × 3 → Espace
2. Hold × 3 → Espace
3. Tap × 3 → Valider

## 📱 Phrases Courtes

### "BONJOUR"
```
-... --- -. .--- --- ..- .-.
B    O  N  J    O  U  R
```

### "MERCI"
```
-- . .-. -.-. ..
M  E R   C    I
```

### "AU REVOIR"
```
.- ..- / .-. . ...- --- .. .-.
A  U  ⎵ R   E V    O  I  R
```
*Note : Le "/" représente un espace entre mots (pas implémenté, utilisez ESPACE)*

### "CA VA"
```
-.-. .- / ...- .-
C    A  ⎵ V    A
```

### "JE T'AIME"
```
.--- . / - .----. .- .. -- .
J    E ⎵ T '     A  I  M  E
```

## 💡 Astuces Pratiques

### Lettres les plus fréquentes (en français)
```
E = .          (la plus courante !)
A = .-
S = ...
I = ..
T = -
N = -.
R = .-.
U = ..-
L = .-..
O = ---
```

### Chiffres faciles à retenir
```
1 = .---- (1 tiret à droite)
2 = ..--- (2 tirets à droite)
3 = ...-- (3 tirets à droite)
4 = ....- (4 tirets à droite)
5 = ..... (5 points, milieu)
6 = -.... (symétrique de 4)
7 = --... (symétrique de 3)
8 = ---.. (symétrique de 2)
9 = ----. (symétrique de 1)
0 = ----- (5 tirets)
```

## 📊 Statistiques Morse

### Durée moyenne par lettre
- Lettres courtes (E, T) : 1-2 secondes
- Lettres moyennes (A, I, N) : 2-3 secondes
- Lettres longues (Q, Y, Z) : 3-5 secondes

### Vitesse de frappe débutant
- Mots par minute : 2-5
- Lettres par minute : 10-25

### Vitesse de frappe expert
- Mots par minute : 15-25
- Lettres par minute : 75-125

## 🔤 Mnémotechniques

### Pour mémoriser les lettres

**A** `.-` : "A-mour" (court puis long)
**B** `-...` : "Bon-jour-nal-iste" (long puis 3 courts)
**C** `-.-.` : "Ça-va-co-mment" (alternance)
**D** `-..` : "Do-mi-no" (long puis 2 courts)
**K** `-.-` : "Kan-gu-rou" (long-court-long)
**O** `---` : "Ohhhh" (3 longs = surprise prolongée)
**S** `...` : "Sa-lu-ta-tions" (3 courts)
**U** `..-` : "U-ni-vers" (2 courts puis long)

## 🎯 Mini-Jeux

### Jeu 1 : Morse Golf
**Règle** : Écrire un mot avec le minimum d'erreurs
**Scoring** : Chaque correction = +1 point (objectif : 0)

### Jeu 2 : Course Contre la Montre
**Règle** : Écrire "HELLO WORLD" le plus vite possible
**Record débutant** : < 2 minutes
**Record expert** : < 30 secondes

### Jeu 3 : Morse Aveugle
**Règle** : Écrire en regardant ailleurs
**Objectif** : Se fier uniquement aux vibrations

### Jeu 4 : Téléphone Morse
**Règle** : Deux personnes s'envoient des messages en Morse
**Difficulté** : Comprendre le Morse de l'autre !
