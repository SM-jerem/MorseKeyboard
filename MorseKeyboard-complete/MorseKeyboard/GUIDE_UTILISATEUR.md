# 👤 Guide Utilisateur - Clavier Morse

## 🎯 Introduction

Le Clavier Morse vous permet d'écrire du texte en utilisant le code Morse international. 
Simple et intuitif, il transforme vos appuis courts et longs en lettres !

---

## 📥 Installation et Activation

### Étape 1 : Installer l'application
1. Transférer l'APK sur votre téléphone
2. Ouvrir le fichier `app-debug.apk`
3. Autoriser l'installation depuis des sources inconnues si demandé
4. Appuyer sur **Installer**

### Étape 2 : Activer le clavier
1. Ouvrir l'application **Clavier Morse**
2. Lire les instructions affichées
3. Appuyer sur **"Ouvrir les Paramètres"**
4. Dans la liste, activer **Clavier Morse** en cochant la case
5. Accepter l'avertissement de sécurité

### Étape 3 : Sélectionner le clavier
1. Ouvrir n'importe quelle application (Messages, Notes, WhatsApp...)
2. Appuyer sur un champ de texte pour faire apparaître le clavier
3. Appuyer sur l'icône **🌐** (globe) ou **⌨️** en bas du clavier
4. Sélectionner **Clavier Morse**

## 🎮 Comment utiliser

### 1. Le bouton MORSE (bleu, grand)

- **Appui COURT** (rapide) → Ajoute un **point** (.)
- **Appui LONG** (maintenir) → Ajoute un **tiret** (-)

💡 **Astuce** : La limite est à 0,25 seconde (250ms)

### 2. Le bouton ESPACE

- Sépare les lettres entre elles
- Obligatoire entre chaque lettre !

### 3. Le bouton VALIDER (vert)

- Convertit tout le Morse en texte
- Insère le texte dans l'application
- Efface le buffer pour recommencer

### 4. Le bouton EFFACER ⌫ (rouge)

- Supprime le dernier symbole (point ou tiret)
- Utile en cas d'erreur

## 📝 Exemple pratique : Écrire "SOS"

Le message de détresse international !

### SOS = ... --- ...

1. **S** = `...` (3 points)
   - Tap rapide
   - Tap rapide
   - Tap rapide
   - Appuyer sur ESPACE

2. **O** = `---` (3 tirets)
   - Maintenir
   - Maintenir
   - Maintenir
   - Appuyer sur ESPACE

3. **S** = `...` (3 points)
   - Tap rapide
   - Tap rapide
   - Tap rapide
   - Appuyer sur VALIDER

✅ Résultat : "SOS" apparaît dans votre texte !

## 💡 Conseils d'utilisation

### Pour débuter
1. **Commencez lentement** : Prenez le temps de bien différencier court/long
2. **Pratiquez les lettres simples** : E (.), T (-), S (...), O (---)
3. **N'oubliez pas les espaces** : Chaque lettre doit être séparée !
4. **Vérifiez la prévisualisation** : Elle vous montre le résultat avant validation

### Pour progresser
- **Mémorisez les lettres courantes** : E, A, T, I, N, S, R, O
- **Utilisez un rythme régulier** : Tapez comme une mélodie
- **Entraînez-vous** : Écrivez votre nom, puis des mots simples

### Erreurs courantes
❌ Oublier le bouton ESPACE → Les lettres se mélangent
❌ Appuyer trop vite sur VALIDER → Pensez à vérifier la prévisualisation
❌ Confondre court et long → Prenez votre temps !

## ❓ Questions Fréquentes

### Q : Mon téléphone ne vibre pas
**R** : Vérifiez que les vibrations ne sont pas désactivées dans les paramètres système.

### Q : Je fais souvent des erreurs court/long
**R** : C'est normal au début ! Avec la pratique, vos doigts vont apprendre le bon timing.

### Q : Puis-je utiliser le clavier pour les mots de passe ?
**R** : Non, Android bloque les claviers personnalisés pour les champs de mot de passe (sécurité).

### Q : Le "?" apparaît dans mon texte
**R** : Cela signifie que vous avez entré une combinaison Morse invalide. Vérifiez la table Morse !

### Q : Comment revenir à mon clavier normal ?
**R** : Appuyez sur l'icône 🌐 ou ⌨️ en bas du clavier, puis sélectionnez votre clavier habituel.

### Q : Puis-je utiliser le clavier dans toutes les applications ?
**R** : Oui ! Messages, WhatsApp, email, navigateur, réseaux sociaux, etc.
